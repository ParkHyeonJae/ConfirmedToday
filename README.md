# ConfirmedToday
오늘 확진자, 실시간 스팀 게임 순위를 집계해주는 프로그램

서버 : C#, 크롤링은 HtmlAgilityPack 패키지를 사용

클라이언트 : C#, C++

## 확진자 현황
- ``!확진자`` 명령어를 통해서 집계가능

<a href="https://imgbb.com/"><img src="https://i.ibb.co/BgS32fx/image.png" alt="image" border="0"></a>

## 스팀게임 순위 ( 1위 ~ 100위 )
- ``!스팀게임순위:startRank:EndRank`` 형식으로 집계가능 (0 ~ 99까지 가능함)
